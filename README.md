# weather_app

Weather App - Aplikasi Informasi dan Riwayat Cuaca Kota Berbasis Server (UAS)

Project Flutter lengkap ini siap diupload sebagai ZIP ke FlutLab.io
(pilih "Upload as a ZIP file" saat membuat project baru).

Sebelum menjalankan, sesuaikan dulu di `lib/main.dart`, pada class
`ApiConfig`, nilai berikut:

- `openWeatherApiKey` — API key dari https://home.openweathermap.org
- `serverListUrl`, `serverSaveUrl`, `serverUpdateUrl`, `serverDeleteUrl`
  — URL REST API di server hosting Anda
